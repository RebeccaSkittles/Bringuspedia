# Gaming on a Receipt Printer

Rating: 4/5

YouTube: https://www.youtube.com/watch?v=oEqvYXYI56s&t=1679s

In this video, Bringus Studios experiments with running simple games on a tiny receipt printer. The project is a playful hack demonstrating how to convert simple graphics into printable frames and stream them to the printer, creating a GIF-like experience.

Key points covered:
- Hardware used (receipt printer model)
- Software to convert images
- Code snippets and examples
